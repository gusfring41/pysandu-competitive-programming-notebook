\subsection{Build System (Makefile)}

Para compilar qualquer arquivo (ex: \texttt{make A} para o arquivo \texttt{A.cpp}) com flags de debug, \textit{sanitizers} e a macro \texttt{LOCAL} ativa, utilize o seguinte \texttt{Makefile}:

\begin{lstlisting}[language=make]
# Variaveis implicitas do Make
CXX = g++
CXXFLAGS = -Wall -Wconversion -Wfatal-errors -g -std=c++17 
 -DLOCAL -fsanitize=undefined,address
LDFLAGS = -fsanitize=undefined,address

\end{lstlisting}
